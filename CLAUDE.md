# Claude Code Notification Plugin

This plugin provides notification hooks for Claude Code. When Claude Code needs approval or completes a task, you'll receive notifications through your preferred channels.

## Supported Notification Methods

- Desktop notifications (Linux/macOS/Windows)
- Slack
- Discord
- Telegram
- WeCom (Enterprise WeChat)
- DingTalk
- Feishu (Lark)
- Custom webhooks
- Sound alerts

## Quick Start

Run the installer:
```bash
./install.sh
```

Then edit `~/.claude/notifications/config.json` to configure your preferred notification methods.
